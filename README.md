# My Clasp Project Appscript

## Steps to use

- Install clasp globally

```cmd
npm install -g @google/clasp
```

- Log in to your Google account

```cmd
clasp login
```

or

```cmd
make login
```

- Then, make necessary change and after all changes done, push code to appscript

```cmd
clasp push
```

or

```cmd
make push
```
